Layout Grid

Criando um template de com grid layout

O modelo gri layout oferece um sistema de layouts em grid 'grade', com linhas e colunas, facilitando desing das páginas we, sem necessidade de usar o float

Um elemento HTML se torna um container de grade quando seu display propriedade é definida como grid ou inline-gri.

Todos os filhos dirertos do container de grade se tornatam automaticamento itens de grande

As linhas verticais dos itens das grades são chamados de colunas, facilitando o layout das páginas.

Todos os itens podem ter um estilo diferente, com isso pode fazer um estilos para para cada item.

As linahs horizontais dos itens das grade são cahamdo de linhas.

Grid Gaps, os espaço entre casa coluna/ linha são chamado de gaps. També pode ajustar o tamanho da lacuna usando uma das seguinte propriedades, coum-gap, row-gap, gap
