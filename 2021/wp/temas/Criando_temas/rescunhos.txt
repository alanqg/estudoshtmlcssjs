Existem duas formas para chamar o arquivo header.php é footer, a primeiro forma que foi aprendido é a 
<?php get_header(); ?>, mesma comando, mas mudando nome do arquivo serve para o arqvuivo footer, <php get_footer(); >?
Segundo forma de faze a chamada dos arquivos é a seguinte <?php include_once 'header.php'; ?>, mesmo comando mudando somente o nome do arquivo para foote.php <?php includ_once 'footer.php'; ?>, todos esses comando não chamados do arquivo general-template.php (Depois estudar cada comando dentro do arquivo).
Comando para a linguagem em: 
<html lang="<?php language_attributes():?>"> (Todo os comando foram explciado, nossa essa configuração de linguagem está dentro do arquivo general-template)
